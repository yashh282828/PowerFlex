# PowerFlex — Gym & Fitness Website

PowerFlex is a modern, premium, fully responsive gym/fitness website built using **HTML**, **CSS**, and **JavaScript**.

## Features
- Dark black/red theme with glassmorphism cards and premium typography
- Sticky navbar with mobile menu
- Smooth scrolling to sections + active section highlighting
- Scroll reveal animations
- Hero section with CTA buttons
- About, Membership plans, Trainers, Testimonials, Contact, Footer
- BMI calculator with metric/imperial toggle, validation, and category result
- Testimonials slider (dots + autoplay + controls)
- Contact form validation with success state

## Project Structure
```text
.
├─ index.html
├─ style.css
└─ script.js
```

## How To Run (Local)

### Option 1: Live Server (VS Code)
1. Install the **Live Server** extension
2. Right-click `index.html` → **Open with Live Server**

### Option 2: Open Directly
Open `index.html` in your browser (works fine for this project).

## Deploy
This is a static site, so you can deploy it to:
- GitHub Pages
- Vercel
- Netlify

## Notes
- The contact form uses client-side validation and shows a success message (no backend).
- BMI is a quick indicator, not a medical diagnosis.
